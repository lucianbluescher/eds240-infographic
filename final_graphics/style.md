## Color palette

Hex codes for all final graphics (keep consistent across plots). **Exception:** 01_sun.qmd keeps its original orange/yellow gradient.

- **Carbon Black**: `#222222`
- **White**: `#ffffff`
- **Baltic Blue**: `#1c5d99`
- **Pacific Blue**: `#639fab`
- **Powder Blue**: `#bbcde5`

### Example (R)

```r
pal <- c(
  carbon_black = "#222222",
  white        = "#ffffff",
  baltic_blue  = "#1c5d99",
  pacific_blue = "#639fab",
  powder_blue  = "#bbcde5"
)
```
